Python function to convert a 1D array into a 2D array .

1st we imports the numpy library.
then we define a function to Converts a 1D array into a 2D array 
with the specified number of rows and columns using NumPy.



Example:-

input ==  convert_to_2d_array([1, 2, 3, 4, 5, 6], 2, 3)
output ==array([[1, 2, 3],
       [4, 5, 6]])





input == convert_to_2d_array([1, 2, 3, 4, 5, 6], 3, 2)
output ==
array([[1, 2],
       [3, 4],
       [5, 6]])
